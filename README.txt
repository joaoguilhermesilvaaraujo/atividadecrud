
1° Utilizar algum IDL para rodar o PHP, recomendo WAMPP / XAMPP

2° O CRUD foi feito em base SQL, insira as tabelas "Tasks.sql" e "userss.sql" em seu MYSQL.

3° O Codigo do arquivo "HMM.PHP" realiza a busca na api jsonplaceholder e salva o codigo em PHP. Você pode carregar a base automaticamente clicando no botão "CARREGAR DB - JSONPLACEHOLDER" no Index.php

4° O arquivod de conexão com o banco de dados é o conn.php verifique a conexão o BANCO DE DADOS está com o nome pré-definido para CRUD, caso use outro altere no arquivo conn.php
